Tutorial

A tutorial is a method of transferring knowledge, often interactive, that teaches by example and practice. You can download and install the tutorial by entering the following command. 

php app.php _develop/git command=submodule/add type=module name=tutorial

You can also install it directly using Git commands without using the framework feature. 

git submodule add https://github.com/TomoakiNagahara/op-module-tutorial.git asset/module/tutorial
